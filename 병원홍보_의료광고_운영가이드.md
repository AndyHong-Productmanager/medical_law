# 병원 홍보 콘텐츠 운영 가이드

## — 대한민국 의료법·의료광고 심의 기준 기반 —

> **최종 업데이트:** 2026년 3월  
> **근거 법령:** 의료법 제56조,제24조, 보건복지부 「유형별 의료광고 사례 및 체크리스트(2판, 2025)」  


## 1. 핵심 법령 요약

### 1-1. 의료광고의 정의 (의료법 제56조 제1항)

의료인등(의료기관 개설자·의료기관의 장·의료인)이 **신문·잡지·음성·음향·영상·인터넷·인쇄물·간판** 등을 통해 의료행위·의료기관·의료인등에 대한 정보를 소비자에게 나타내거나 알리는 행위.

### 1-2. 사전심의 대상 (의료법 제57조)

| 매체 유형 | 사전심의 필요 여부 |
|---|---|
| 신문·인터넷신문·정기간행물 | ✅ 필요 |
| 옥외광고물(현수막·벽보·전단 등) | ✅ 필요 |
| **일평균 이용자 10만 명 이상 SNS 플랫폼** (네이버, 카카오, 유튜브, 인스타그램, 틱톡 등) | ✅ 필요 |
| 병원 자체 홈페이지 (직접 도메인) | ❌ 면제 (단, 의료법 제56조 금지사항은 준수) |

> **주의:** 네이버 블로그를 홈페이지 대용으로 사용해도, 네이버 플랫폼 자체가 10만 명 이상이므로 **사전심의 대상**이다. 이것이 보건복지부의 공식 해석이다.

### 1-3. 심의 없이 가능한 최소 정보 (의료법 제57조 제3항)

다음 사항**만**으로 구성된 광고는 심의가 면제된다:

- 의료기관 명칭·소재지·전화번호
- 진료과목
- 소속 의료인 성명·성별·면허 종류
- 기타 대통령령이 정하는 사항

### 1-4. 절대 금지 광고 유형 (의료법 제56조 제2항)

1. 신의료기술평가를 받지 않은 기술에 대한 광고
2. **치료경험담** 등 치료 효과 오인 우려가 있는 광고
3. 거짓·과장 광고
4. 다른 의료인·의료기관과 비교하여 우수하다는 광고
5. 비급여 진료비를 할인·면제한다는 광고
6. 전문병원 지정을 받지 않았으면서 전문병원 명칭 사용
7. 각종 인증·보증을 부당 표시하는 광고

### 1-5. 위반 시 처벌

| 위반 유형 | 처벌 기준 |
|---|---|
| 의료법 제56조 위반 (금지 광고) | 1년 이하 징역 또는 1,000만 원 이하 벌금 |
| 행정처분 | 업무정지 1~2개월 |
| 환자 유인·알선 (제27조 제3항) | 3년 이하 징역 또는 3,000만 원 이하 벌금 |

## 2. 여섯 가지 핵심 질문과 답변

### Q1. 단순 의학지식 전달 — 광고인가?

**원칙적으로 광고가 아니다.** 보건복지부(보건의료정책과-424, 2025.1.16.)는 다음과 같이 회신했다:

> *"일반적인 의학정보나 의료기관 또는 의료인의 단순 근황 등을 정보 제공의 목적으로 게시하는 경우 일률적으로 의료법상 의료광고에 해당한다고 보기는 어려움."*

**다만**, 개별 게시물이 의료광고인지는 **목적·내용·게시형태·타 게시물과의 연관성** 등을 종합 판단한다. 즉 단순 정보라도 다음 요소가 결합되면 광고로 판단될 수 있다:

- 병원명·연락처가 반복 노출
- 특정 시술로의 유도 링크나 예약 CTA가 포함
- "OO의원만의 노하우" 같은 광고성 문구 삽입
- 시리즈 형태로 특정 시술을 집중 홍보

**안전한 정보 콘텐츠의 기준:**

- 질병의 원인·증상·예방법 등 일반적 의학 상식
- 특정 병원이 아닌 의학 전체에 적용되는 보편적 내용
- "전문의와 상의하세요" 수준의 권고
- 출처(학회·논문·가이드라인)를 명시한 학술적 정보

### Q2. "우리 병원이 잘한다 / 특화되었다" — 말할 수 있는가?

**매우 제한적으로만 가능하다.**

#### ✅ 가능한 표현

- 진료과목, 세부 진료분야 안내 (예: "통증의학과 진료")
- 의료진의 **객관적 경력** (학력, 학회 임원, 수련 병원 등 — 단, 6개월 초과 임상경력에 한함)
- 보유 장비·시설의 **사실적 나열** (과장 없이)
- 보건복지부 지정 전문병원인 경우 해당 사실 표시

#### ❌ 금지되는 표현

| 금지 표현 예시 | 위반 조항 |
|---|---|
| "OO 분야 최고 / 국내 1위 / 최다" | 비교·과장 광고 |
| "반드시 효과가 있다 / 완치된다" | 치료 효과 보장 |
| "타 병원과 달리 우리만의 기술" | 비교 광고 |
| "OO 전문병원" (미지정 상태) | 전문병원 명칭 위반 |
| "#성형 #전문병원" (해시태그 포함) | 전문병원 명칭 위반 |
| "00건 시술 경험" (6개월 이하 경력) | 단기 임상경력 광고 |

#### 💡 실전 팁

"특화"라는 단어 자체를 쓸 수 있느냐가 아니라, **맥락에서 비교 우위를 암시하는지**가 판단 기준이다. "통증 치료에 집중하는 의원입니다" 정도는 사실의 기술이지만, "통증 치료 특화 전문의원"은 전문병원 명칭 사용으로 오해받을 수 있다.

### Q3. 환자 치료후기(치료경험담)를 넣어도 되는가?

**가장 민감한 영역이다. 구분해서 이해해야 한다.**

#### (A) 환자가 자발적으로 올리는 후기 → 규제 대상 아님

의료법은 **의료인등이 하는 광고**만 규제한다. 환자가 금전적 대가 없이, 병원의 사주 없이 개인 블로그·카페·SNS에 자발적으로 올리는 후기는 의료광고가 아니다.

#### (B) 병원이 직접 올리거나 대가를 제공한 후기 → 의료법 위반

- 병원 직원 ID로 작성한 후기
- 인플루언서에게 대가(시술비 할인, 현금 등)를 주고 올리게 한 후기
- 체험단에게 진료비 할인 혜택을 제공하고 받은 후기

이것은 **"치료경험담을 사용한 의료광고"** (의료법 제56조 제2항 제2호)에 해당하며, 절대 금지된다.

#### (C) 병원 홈페이지에 치료경험담을 게재하는 경우

보건복지부 유권해석(2018)에 따르면, **로그인 절차를 통해 불특정 다수가 열람할 수 없는 공간**에 게재된 경우에 한해 '광고'로 보기 어렵다고 해석한 바 있다.

**단, 다음 경우는 여전히 광고로 본다:**
- 소셜 로그인(네이버/카카오 간편 로그인)으로 누구나 쉽게 접근 가능한 경우
- 임시 아이디 발급으로 실질적 접근 제한이 없는 경우

#### 💡 실무 권고

**치료경험담은 아예 사용하지 않는 것이 가장 안전하다.** 부득이 사용해야 한다면, 병원 자체 홈페이지에 **실질적 로그인 장벽**(회원가입 필수)을 설치한 비공개 게시판에만 게재한다.

### Q4. 비포 & 애프터(치료 전·후 사진) — 어디에 넣을 수 있는가?

**"치료 전·후 사진"과 "치료경험담"은 법적으로 구분되는 개념이다.** 이 구분이 핵심이다.

#### 보건복지부 공식 견해

보건복지부는 질의 응답을 통해 다음과 같이 밝혔다:

> **치료 전·후 사진 자체는 "치료경험담"이 아니다.** 따라서 조작을 가하지 않은 치료 전·후 사진은 로그인 기능 없이도 의료광고에 활용할 수 있다.

#### 사용 시 필수 준수사항

| 항목 | 기준 |
|---|---|
| 환자 동의서 | 반드시 확보 (의광심위 양식 사용 권장) |
| 촬영 조건 | 크기·각도·조명·메이크업·배경이 동일 |
| 경과기간 | **3개월 이상** 경과된 사진만 허용 |
| 시술 명칭 | 구체적인 시술/수술 명칭 표시 필수 |
| 경과기간 표시 | 시술 후 몇 개월 경과인지 명시 필수 |
| 부작용 표시 | 해당 시술별 2~3개 이상의 부작용을 **사진 아래에** 본문과 동일 크기로 표시 |
| 원본 사진 | 환자 본인 확인 가능한 원본 제출 (심의 시) |

#### 매체별 게재 가능 여부

| 매체 | 전·후 사진만 | 전·후 사진 + 치료경험담 |
|---|---|---|
| 병원 자체 홈페이지 (로그인 없이) | ✅ 가능 (부작용 표시 필수) | ❌ 경험담은 로그인 구간에만 |
| 병원 자체 홈페이지 (로그인 후) | ✅ 가능 | ⚠️ 가능하나 로그인이 실질적이어야 |
| 네이버 블로그 | ⚠️ 사전심의 필요 | ❌ 경험담 포함 시 위법 |
| 인스타그램 | ⚠️ 사전심의 필요 | ❌ 경험담 포함 시 위법 |
| 유튜브 | ⚠️ 사전심의 필요 | ❌ 경험담 포함 시 위법 |

### Q5. B&A는 무조건 로그인 공간에만 해야 하는가? — 사실관계 정리

이 부분에 혼선이 많은 이유는 **의료광고심의위원회의 심의기준**과 **보건복지부의 유권해석**이 다르기 때문이다.

#### 의료광고심의위원회 심의기준 (2019년 도입)

심의위원회는 다음과 같이 규정했다:

- 심의 대상 매체에는 치료 전·후 사진을 게재할 수 없다
- 홈페이지에서는 **로그인 기능을 만들어 후기와 함께 숨겨놓아야** 한다

#### 보건복지부 질의 응답 (이후 회신)

보건복지부는 심의위원회와 **다른 견해**를 밝혔다:

> **치료 전·후 사진은 그것이 곧바로 "치료경험담(후기)"에 해당하지 않으므로, 로그인 기능 없이 의료광고에 활용할 수 있다.**

또한 보건복지부 가이드라인(2판, 2025)에서도, **진료한 환자의 치료사례 사진은 시기가 명확하고 별도 조작 처리가 되지 않았다면 로그인 절차 없이 게재가 가능**하다고 안내했다.

#### 결론: 현재의 법적 상황

| 구분 | 로그인 필요 여부 |
|---|---|
| **치료 전·후 사진만** (사진 + 시술명 + 경과기간 + 부작용) | ❌ 로그인 불필요 |
| **치료경험담** (환자의 주관적 경험·감상·추천) | ✅ 로그인 필수 |
| **사진 + 경험담 결합** | ✅ 로그인 필수 (경험담이 포함되므로) |

> **실무 주의사항:** 보건복지부의 견해가 심의위원회보다 상위 해석이나, 지자체 보건소마다 단속 기준이 다를 수 있다. 가장 안전한 방법은 **사진만 별도로 분리하고, 주관적 경험담은 절대 함께 넣지 않는 것**이다. 사진 아래에는 시술명·경과기간·부작용만 기재한다.

### Q6. 블로그/SNS 안전 안내문구 — 어떻게 넣는가?

블로그나 SNS 게시물에 안내문구(면책고지)를 넣는다고 의료법 위반이 면제되는 것은 아니다. 그러나 **선의의 정보 제공 목적을 입증하는 보조 자료**로 활용할 수 있다. 다음은 상황별 권장 문구다.

#### (A) 정보성 콘텐츠용 안내문구 (글 상단)

```
ℹ️ 본 게시물은 의학 정보를 제공하기 위한 목적으로 작성되었으며,
특정 의료기관의 의료광고가 아닙니다.
개인의 증상과 상태에 따라 치료 방법이 달라질 수 있으므로,
반드시 전문의와 상담 후 결정하시기 바랍니다.
```

#### (B) 정보성 콘텐츠용 안내문구 (글 하단)

```
📋 안내
본 콘텐츠는 질병 예방 및 건강 증진을 위한 일반적인 의학 정보입니다.
의료법 제56조에 따른 의료광고에 해당하지 않으며,
특정 치료법의 효과를 보장하는 내용이 아닙니다.
증상이 있으신 분은 가까운 의료기관을 방문하여 전문의의 진료를 받으시기 바랍니다.
```

#### (C) 치료 전·후 사진 게시 시 필수 병기 문구

```
⚠️ 치료 결과는 개인에 따라 차이가 있을 수 있습니다.
시술명: [구체적 시술명]
시술 후 경과기간: [O개월]
부작용: [해당 시술의 주요 부작용 2~3가지]
본 사진은 환자 본인의 동의하에 게재되었습니다.
```

#### (D) 심의필 광고 게시물 필수 표시

```
[의료광고심의필] 심의번호: 제OOOO-OOOOOO호
유효기간: OOOO.OO.OO ~ OOOO.OO.OO
```

#### (E) 유튜브 영상용 안내문구 (영상 설명란)

```
📋 본 영상은 의학 정보 제공을 목적으로 제작되었으며,
의료법 제56조에 따른 의료광고에 해당하지 않습니다.
영상 내 정보는 일반적인 의학 지식에 기반하며,
개인의 건강 상태에 따라 적용이 다를 수 있습니다.
정확한 진단과 치료를 위해 반드시 전문의와 상담하시기 바랍니다.
```

## 3. 채널별 운영 규칙

### 3-1. 병원 자체 홈페이지 (직접 도메인)

**사전심의 면제 매체** — 그러나 의료법 제56조의 금지사항은 그대로 적용된다.

| 허용 | 금지/주의 |
|---|---|
| 의료기관 정보 (명칭, 주소, 진료시간, 진료과목) | 거짓·과장 표현 |
| 의료진 경력 (6개월 초과 임상경력) | "최고" "1위" 등 비교 표현 |
| 보유 장비·시설 사실적 안내 | 치료 효과 보장 문구 |
| 치료 전·후 사진 (부작용 표시 필수) | 비급여 할인 광고 |
| 로그인 구간 내 치료경험담 | 로그인 없는 치료경험담 |
| 일반 의학정보 콘텐츠 | 대가성 후기 게재 |

**권장 구조:**

```
홈페이지 메인
├── 병원 소개 (의료진, 장비, 시설) — 공개
├── 진료 안내 (진료과목, 질환 정보) — 공개
├── 건강 정보 (의학 칼럼, 질병 예방) — 공개
├── 치료 사례 (전·후 사진) — 공개 (부작용 필수 표시)
└── 환자 후기 — 회원 로그인 후 열람 (실질적 접근 제한)
```

### 3-2. 블로그 (네이버 블로그 등)

**사전심의 대상 매체** — 10만 명 이상 플랫폼이므로.

#### 게시 가능한 콘텐츠

- 질병·증상·예방에 대한 일반 의학정보 (병원 홍보 요소 최소화)
- 의료계 동향, 건강 뉴스 큐레이션
- 의료인의 일상, 원내 행사 등 근황

#### 게시 시 주의가 필요한 콘텐츠

- 특정 시술/장비 소개 → 사전심의 후 심의필 표시
- 치료 전·후 사진 → 사전심의 + 부작용 표시
- 이벤트·프로모션 → 비급여 할인으로 해석될 위험

#### 절대 금지

- 심의 없는 의료광고 게시물
- 치료경험담 (대가 유무 불문, 병원이 게시 주체인 경우)
- 비급여 가격 할인·면제 정보
- 전문병원 미지정 상태에서 전문병원 명칭 사용

#### 블로그 게시물 구조 템플릿

```
[상단 안내문구]
ℹ️ 본 게시물은 의학 정보 제공 목적으로 작성되었으며,
특정 치료법의 효과를 보장하지 않습니다.

[본문 — 질환/증상/예방 중심 정보]

[하단 안내문구]
📋 본 콘텐츠는 의료법 제56조에 따른 의료광고가 아닌
건강 정보 제공 목적의 게시물입니다.
개인의 증상에 따라 차이가 있을 수 있으며,
정확한 진단과 치료는 전문의 상담을 통해 결정하시기 바랍니다.

[의료기관 기본정보: 기관명·주소·전화번호 — 이 정보는 심의 면제 대상]
```

### 3-3. 유튜브

**사전심의 대상 매체.**

#### 영상 유형별 판단

| 영상 유형 | 광고 해당 여부 | 조치 |
|---|---|---|
| 의학 교육 영상 (질병 설명, 예방법) | 일반적으로 광고 아님 | 안내문구 삽입 권장 |
| 시술 과정 영상 (특정 병원 시술) | 광고에 해당할 가능성 높음 | 사전심의 필요 |
| 의사 브이로그 (일상+진료 혼합) | 내용에 따라 개별 판단 | 광고성 요소 최소화 |
| 환자 인터뷰/후기 영상 | 치료경험담 광고 | ❌ 금지 |
| 시술 전·후 비교 영상 | 사전심의 대상 | 부작용 표시 필수 |

#### 유튜브 운영 핵심 원칙

1. 영상 제목에 "후기" "체험" "경험담" 등의 표현을 넣지 않는다
2. 영상 설명란에 안내문구를 반드시 포함한다
3. 썸네일에 전·후 사진을 사용할 경우 부작용 문구 병기가 어려우므로 가급적 피한다
4. 심의필 광고 영상은 심의번호와 유효기간을 영상 설명란과 영상 내에 표시한다

### 3-4. SNS (인스타그램·틱톡 등)

**사전심의 대상 매체.**

#### 콘텐츠 가이드

| 콘텐츠 유형 | 허용 여부 | 비고 |
|---|---|---|
| 건강 팁 카드뉴스 | ⚠️ 내용에 따라 | 병원 홍보 요소 최소화 |
| 의료진 소개 (학력·경력) | ✅ 가능 | 사실에 기반한 정보 |
| 원내 시설·분위기 사진 | ✅ 가능 | 시술 유도 문구 없이 |
| 시술 전·후 사진 | ⚠️ 사전심의 | 부작용 필수 표시 |
| 리그램/공유된 환자 후기 | ❌ 금지 | 병원이 공유한 순간 광고 주체 |
| 해시태그 "#전문병원" | ❌ 금지 | 전문병원 미지정 시 위반 |
| DM으로 진료비 할인 안내 | ❌ 금지 | 비급여 할인 광고 |

증상이 있으시면 전문의 상담을 받으시기 바랍니다.

## 4. 자주 하는 실수와 대응

| 실수 유형 | 리스크 | 대응 |
|---|---|---|
| 블로그에 "후기" 제목으로 전·후 사진 + 경험담 게재 | 치료경험담 광고 위반 | 경험담 삭제, 사진만 부작용 표시하여 재게시 (심의 후) |
| 인스타 릴스에 시술 영상 + "효과 대박" 자막 | 치료 효과 보장 + 미심의 | 즉시 삭제 또는 비공개 전환 |
| 네이버 블로그를 홈페이지처럼 운영하며 심의 없이 광고 | 사전심의 위반 | 광고성 게시물 전체 비공개 후 심의 진행 |
| 환자에게 할인 제공 후 후기 작성 요청 | 대가성 치료경험담 + 환자유인 | 즉시 중단, 기존 게시물 삭제 |
| "전문병원" 해시태그 사용 (미지정) | 전문병원 명칭 위반 | 해시태그 즉시 삭제 |
| 심의받은 광고의 일부 내용을 수정하여 게시 | 심의 내용 변경 → 미심의 | 원본대로 복원하거나 재심의 |

## 5. 안전한 콘텐츠 전략 — 정리

### "정보성 콘텐츠"와 "광고성 콘텐츠"를 완전 분리하라

**정보성 콘텐츠 (심의 불필요 영역)**
- 질환 교육, 예방법, 생활습관 가이드
- 의료계 동향, 학회 소식
- 의사 칼럼, 의학 Q&A
- 원내 소식, 의료진 근황

**광고성 콘텐츠 (심의 후 게재)**
- 특정 시술·장비 홍보
- 전·후 사진 활용 콘텐츠
- 이벤트·프로모션 (비급여 할인은 원천 금지)

### 채널별 최적 전략

| 채널 | 주력 콘텐츠 | 핵심 주의점 |
|---|---|---|
| 자체 홈페이지 | 전체 (정보+광고+후기) | 후기는 로그인 구간에만 |
| 네이버 블로그 | 정보성 콘텐츠 위주 | 광고성은 반드시 심의 후 |
| 유튜브 | 의학교육·브이로그 | 후기 영상 금지, 시술 영상은 심의 |
| 인스타그램 | 건강 팁·시설 소개 | 릴스 전·후 비교 주의 |
| 틱톡 | 건강 상식 숏폼 | 짧은 영상이라도 광고면 심의 |
